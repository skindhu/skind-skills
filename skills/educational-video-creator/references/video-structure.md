## General Structure

Every educational video follows this pattern:

```
┌─────────────────────────────────────────┐
│  1. HOOK         - Grab attention       │
│  2. INTRO        - Set expectations     │
│  3. CONTENT      - Main teaching        │
│  4. SUMMARY      - Reinforce learning   │
│  5. OUTRO        - Close and CTA        │
└─────────────────────────────────────────┘
```

## Structure by Duration

### Short Video (1-3 minutes)

```
Total: 90-180 seconds

┌─ Hook (5-10s) ────────────────────────────┐
│  • Surprising fact or question            │
│  • Visual that creates curiosity          │
└───────────────────────────────────────────┘
        ↓
┌─ Introduction (10-15s) ───────────────────┐
│  • Topic statement                        │
│  • Preview of what viewer will learn      │
└───────────────────────────────────────────┘
        ↓
┌─ Main Content (60-120s) ──────────────────┐
│  • 2-3 key points maximum                 │
│  • One concept per segment                │
│  • Visual demonstration for each          │
└───────────────────────────────────────────┘
        ↓
┌─ Summary (10-15s) ────────────────────────┐
│  • Quick recap of key points              │
│  • Single takeaway message                │
└───────────────────────────────────────────┘
        ↓
┌─ Outro (5-10s) ───────────────────────────┐
│  • Thank viewer                           │
│  • Optional: Related topic tease          │
└───────────────────────────────────────────┘
```

**Example Timeline (2 minutes):**
```
0:00 - 0:08  Hook
0:08 - 0:20  Introduction
0:20 - 1:40  Main Content (3 segments × 25-30s)
1:40 - 1:52  Summary
1:52 - 2:00  Outro
```

### Medium Video (3-5 minutes)

```
Total: 180-300 seconds

┌─ Hook (5-15s) ────────────────────────────┐
│  • Engaging question or scenario          │
│  • Relatable problem statement            │
└───────────────────────────────────────────┘
        ↓
┌─ Introduction (15-30s) ───────────────────┐
│  • Topic context                          │
│  • Learning objectives (what you'll learn)│
│  • Brief overview of structure            │
└───────────────────────────────────────────┘
        ↓
┌─ Main Content (120-200s) ─────────────────┐
│  Segment 1: Foundation (30-45s)           │
│    • Basic concept introduction           │
│    • Visual demonstration                 │
│                                           │
│  Segment 2: Deep Dive (45-60s)            │
│    • Detailed explanation                 │
│    • Examples and analogies               │
│                                           │
│  Segment 3: Application (30-45s)          │
│    • Real-world examples                  │
│    • How it connects to viewer's life     │
│                                           │
│  Segment 4: (Optional) Advanced (30-45s)  │
│    • Additional depth for curious viewers │
└───────────────────────────────────────────┘
        ↓
┌─ Summary (20-30s) ────────────────────────┐
│  • Recap each main point                  │
│  • Connect back to opening hook           │
│  • Main takeaway                          │
└───────────────────────────────────────────┘
        ↓
┌─ Outro (10-15s) ──────────────────────────┐
│  • Fun fact or additional insight         │
│  • Encourage exploration                  │
│  • Optional: Call to action               │
└───────────────────────────────────────────┘
```

**Example Timeline (4 minutes):**
```
0:00 - 0:10  Hook
0:10 - 0:30  Introduction
0:30 - 1:15  Segment 1: Foundation
1:15 - 2:15  Segment 2: Deep Dive
2:15 - 3:00  Segment 3: Application
3:00 - 3:25  Summary
3:25 - 4:00  Outro + Fun Fact
```

### Long Video (5-10 minutes)

```
Total: 300-600 seconds

┌─ Hook (10-20s) ────────────────────────────┐
│  • Compelling story opening               │
│  • Problem or mystery to solve            │
│  • Stakes: why this matters               │
└───────────────────────────────────────────┘
        ↓
┌─ Introduction (20-40s) ───────────────────┐
│  • Context and background                 │
│  • Clear learning objectives              │
│  • Roadmap of video structure             │
└───────────────────────────────────────────┘
        ↓
┌─ Main Content (200-450s) ─────────────────┐
│                                           │
│  Chapter 1: Foundation (60-90s)           │
│  ├─ Core concept                          │
│  ├─ Visual explanation                    │
│  └─ Mini-summary                          │
│                                           │
│  Chapter 2: Mechanics (60-90s)            │
│  ├─ How it works                          │
│  ├─ Step-by-step breakdown                │
│  └─ Mini-summary                          │
│                                           │
│  Chapter 3: Implications (60-90s)         │
│  ├─ Real-world applications               │
│  ├─ Examples and case studies             │
│  └─ Mini-summary                          │
│                                           │
│  Chapter 4: (Optional) Advanced (45-60s)  │
│  ├─ Deeper concepts                       │
│  ├─ Edge cases or exceptions              │
│  └─ Mini-summary                          │
│                                           │
└───────────────────────────────────────────┘
        ↓
┌─ Summary (30-45s) ────────────────────────┐
│  • Comprehensive recap                    │
│  • Key insights and takeaways             │
│  • Answer to opening hook/question        │
└───────────────────────────────────────────┘
        ↓
┌─ Outro (15-30s) ──────────────────────────┐
│  • Interesting facts                      │
│  • Further learning suggestions           │
│  • Closing thought/inspiration            │
│  • Credits (if needed)                    │
└───────────────────────────────────────────┘
```

**Example Timeline (7 minutes):**
```
0:00 - 0:15  Hook
0:15 - 0:45  Introduction
0:45 - 2:00  Chapter 1: What are the Four Forces?
2:00 - 3:30  Chapter 2: How Each Force Works
3:30 - 5:00  Chapter 3: Forces in Action
5:00 - 5:45  Chapter 4: Special Cases
5:45 - 6:20  Summary
6:20 - 7:00  Outro + Fun Facts
```

## Section Templates

### Hook Templates

**Question Hook:**
```
"你有没有想过，[surprising fact or question]？"
"Have you ever wondered why [common phenomenon]?"
```

**Scenario Hook:**
```
"想象一下，[relatable scenario that leads to topic]"
"Picture this: [engaging scenario]"
```

**Problem Hook:**
```
"[Topic] 看起来很简单，但其实..."
"[Topic] seems simple, but there's more to it..."
```

### Introduction Templates

```markdown
## Introduction Script Template

"今天，我们要探索 [topic]。
在接下来的 [duration] 里，你会了解到：
- [Learning point 1]
- [Learning point 2]  
- [Learning point 3]
让我们开始吧！"
```

### Summary Templates

```markdown
## Summary Script Template

"让我们回顾一下今天学到的内容：
首先，[Key point 1]。
其次，[Key point 2]。
最后，[Key point 3]。
记住，[Main takeaway message]。"
```

### Outro Templates

```markdown
## Outro Script Template

"下次当你 [related everyday situation]，
想想今天学到的 [topic]。
科学就在我们身边！"

## With Fun Fact
"有趣的是，[surprising related fact]。
感谢观看，我们下次再见！"
```

## Pacing Guidelines

### Information Density

| Video Length | New concepts/minute | Words/minute (narration) |
|--------------|---------------------|--------------------------|
| Short (1-3m) | 1-2 concepts | 120-140 words |
| Medium (3-5m) | 1.5-2 concepts | 130-150 words |
| Long (5-10m) | 1-1.5 concepts | 140-160 words |

### Visual Pacing

- **Scene duration**: 3-8 seconds average
- **Maximum static hold**: 5 seconds
- **Minimum between cuts**: 2 seconds
- **Complex visuals**: Allow 5-10 seconds

### Cognitive Load Management

```
┌─────────────────────────────────────────┐
│  Rule of 3:                             │
│  • Max 3 new concepts per segment       │
│  • Max 3 visual elements animating      │
│  • Max 3 seconds without visual change  │
└─────────────────────────────────────────┘
```

## Transition Recommendations

### Between Segments

| Transition Type | When to Use | Duration |
|-----------------|-------------|----------|
| Fade | Topic change | 0.5-1s |
| Slide | Continuing same topic | 0.3-0.5s |
| Zoom | Focusing on detail | 0.5-0.8s |
| Wipe | New chapter | 0.5-0.8s |

### Within Segments

- Use subtle animations (fade, slide)
- Keep transition duration under 0.5s
- Maintain visual continuity
